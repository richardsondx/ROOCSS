Welcome to ROOCSS : Rich Object Oriented CSS Pages.

ROOCSS make it easier to build stylesheet for medium and large projects. It was build with the intent to improve the development of website on a ruby on rails environment but it can be applied to any type of web language or framework.

Guideline: An Introduction to Rich Object Oriented CSS (ROOCSS)

  • Separation of structure from skin
  • Separation of containers and content
  • Grid set the width and Content set the height
  • CSS with the suffix "_debug" define the look

FAQ

  • Where ROOCSS come from? ROOCSS was inspired by OOCSS created by Nicole Sullivan (https://github.com/stubbornella/). If you have any question about the OOCSS part of ROOCSS just visit https://github.com/stubbornella/oocss/wiki/faq

  • How is ROOCSS different from OOCSS? ROOCSS was designed to fit my need as a Ruby on Rails Developer.

  • Can I remove unused styles? Totally. If you are done building you're website and you don't need some element feel free to remove them.
